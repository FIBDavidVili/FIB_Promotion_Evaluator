# Badge Builder Website

This is a Vite + React + TypeScript version of your badge builder website, ready for GitHub and Vercel.

## Before you run it

Put your badge PNG files in:

public/badges/

Required filenames:

- command.png
- trial-low-command.png
- supervisor.png
- trial-supervisor.png
- patrol-agent.png

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Put it online with Vercel

1. Upload this folder to a GitHub repo.
2. Import the repo into Vercel.
3. Deploy.

Vercel should detect this as a Vite app automatically.
